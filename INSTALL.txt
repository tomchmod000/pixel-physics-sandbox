## Setup

### How to setup Graphics module on MacOS

    1. Install XQuartz
    2. Run [opam install graphics] 
    3. Say yes if opam prompts installation of other packages
    4. Restart your computer
    5. Go to applications -> utilities -> Run XQuartz application

### How to setup Graphics module on WSL (Windows)

    1. Run [opam install graphics] 
    2. Say yes if opam prompts installation of other packages

## How to Build and Run

To build the application, run `make build` in the terminal.

To run the application, run `make play` in the terminal.

## Interacting with the program

Once the game window pops up, press any key to see the functions for drawing / deleting a pixel at work.

Press the exit button on the window to close it.
